DBConnectionManager

A DBConnectionManager é uma classe utilitária para gerenciar conexões de banco de dados com Sequelize, usando o padrão Singleton para diferentes esquemas de conexão. Suporta bancos Oracle e outros dialetos, com configurações definidas em config.js. Depende de Utils.

Métodos

getDefaultDBConnection

Tipo: method

Tipo retorno: Sequelize | null

Qualificadores

  • static

Retorna ou inicializa a conexão padrão Singleton com base na configuração do ambiente.

Exemplo de Uso


  const conn = DBConnectionManager.getDefaultDBConnection();
  if (conn) {
    console.log("Conexão padrão inicializada.");
  }

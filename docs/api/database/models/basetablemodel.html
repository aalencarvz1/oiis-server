BaseTableModel

A BaseTableModel é uma classe base que estende Sequelize.Model para definir modelos de tabelas com campos e métodos comuns. É usada como base para modelos Sequelize, implementando funcionalidades como migrações, associações e operações CRUD. Depende de DBConnectionManager, Utils, DataSwap, e DatabaseUtils.

Propriedades Field (campos de tabela)

Estas propriedades são declaradas com a palavra reservada declare no TypeScript, indicando que correspondem diretamente aos campos da tabela no Sequelize quando o modelo é instanciado.

id

Tipo: number | undefined

Chave primária do registro, auto-incrementada.

parent_id

Tipo: number | undefined

ID do registro pai, para registros hierárquicos.

status_reg_id

Tipo: number

ID do status do registro.

creator_user_id

Tipo: number | undefined

ID do usuário que criou o registro.

created_at

Tipo: Date | undefined

Data de criação do registro.

updater_user_id

Tipo: number | undefined

ID do usuário que atualizou o registro.

updated_at

Tipo: Date | undefined

Data de atualização do registro.

data_origin_id

Tipo: number | undefined

ID da origem externa do registro.

id_at_origin

Tipo: string | undefined

ID do registro na origem externa.

source_id

Tipo: number | undefined

ID da fonte do registro (ex.: registro duplicado).

deleted_at

Tipo: Date | undefined

Data de exclusão lógica do registro.

is_sys_rec

Tipo: number

Indica se o registro é do sistema (0 ou 1).

Inicialização Estática

O bloco estático inicializa a propriedade foreignsKeys chamando getForeignKeys. Isso garante que as chaves estrangeiras do modelo sejam configuradas no momento da definição da classe.

Exemplo de Efeito

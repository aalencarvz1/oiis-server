DataSwap

A DataSwap é uma classe utilitária para encapsular resultados de operações, incluindo status de sucesso, dados, mensagens e exceções. É usada em contextos como respostas de APIs ou operações de banco de dados, frequentemente em conjunto com Utils.

Propriedades

success

Tipo: boolean

Indica se a operação foi bem-sucedida. Inicializado como false.

status

Tipo: number

Código de status da operação. Inicializado como -1.

data

Tipo: any

Armazena os dados retornados pela operação. Inicializado como null.

message

Tipo: string | null

Mensagem descritiva da operação (ex.: erro ou sucesso). Inicializado como null.

exception

Tipo: any

Objeto de exceção associado à operação. Inicializado como null.

Métodos

setDataSwap

Tipo: method

Parâmetros

  • newDataSwap: DataSwap - Outro objeto DataSwap para copiar propriedades.

Tipo retorno: void

Atualiza as propriedades success, data, message, e exception com base em outro objeto DataSwap.

Exemplo de Uso


  const ds1 = new DataSwap();
  const ds2 = new DataSwap();
  ds2.success = true;
  ds2.data = { id: 1 };
  ds1.setDataSwap(ds2);
  console.log(ds1.success); // true
  console.log(ds1.data); // { id: 1 }

setException

Tipo: method

Parâmetros

  • exception: any - Objeto de exceção a ser definido.
  • notConsole?: boolean - Se true, não exibe o erro no console (padrão: false).

Tipo retorno: void

Define uma exceção, atualizando success para false, message com base na exceção, e exception com o objeto fornecido.

Exemplo de Uso


  const ds = new DataSwap();
  ds.setException(new Error("Falha na operação"));
  console.log(ds.success); // false
  console.log(ds.message); // "Falha na operação"
  console.log(ds.exception); // Error: Falha na operação

throw

Tipo: method

Tipo retorno: never

Lança a exceção armazenada em exception ou uma nova Error com base em message.

Exemplo de Uso


  const ds = new DataSwap();
  ds.setException(new Error("Erro crítico"));
  try {
    ds.throw();
  } catch (e) {
    console.log(e.message); // "Erro crítico"
  }

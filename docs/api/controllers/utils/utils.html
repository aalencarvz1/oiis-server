Utils

A Utils fornece métodos utilitários estáticos para manipulação de dados, logging, conversão de tipos, validações e outras funcionalidades, usados por classes como BaseTableModel, DatabaseUtils, e outras.

Propriedades

A classe Utils não possui propriedades explícitas definidas. Todos os membros são métodos estáticos.

Métodos

Tipo: method

Tipo retorno: string

Qualificadores

Retorna a data e hora atual no formato DD/MM/YYYY HH:MM:SS.MMM.

Exemplo de Uso


  Utils.getMomento(); // "06/05/2025 12:34:56.789"
